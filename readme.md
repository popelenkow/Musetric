# Musetric

Musetric is a vocal training application.

## Development

```bash
yarn
yarn dev
```

## Check

```bash
yarn
yarn check:ts
yarn check:lint
yarn check:format
yarn test
```

## Docker

```bash
yarn build:frontend
yarn build:docker
yarn start:docker
```
